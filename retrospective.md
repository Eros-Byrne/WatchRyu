Developing **WatchRyu** was a rewarding journey that deepened my understanding of Modern Android Development. I successfully met all initial goals, including 7-tab navigation, deep analytics, and professional branding.

One major challenge I encountered was the instability of the Jikan API when importing certain accounts (like 'Enigmy'). Even with public settings, MyAnimeList often blocks scraper-based requests, returning a 404 error. I adapted by implementing a more robust error-handling layer and building a reliable manual XML import fallback. This experience taught me that real-world engineering requires anticipating third-party failures and providing users with clear, actionable alternatives.

I am particularly proud of the accessibility system. Implementing global font scaling using custom theme attributes and overlays was technically demanding but resulted in a much more inclusive product. If I had more time, I would implement the official MAL OAuth2 API to allow for "Private" list access and two-way syncing. Overall, the project reflects a professional-tier application that strictly follows industry-driven quality guidelines while providing a high-quality, branded user experience on the Pixel 3a.
