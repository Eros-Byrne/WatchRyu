# Overview

Student ID: `c24056128`

The **Anime Tracker** is a mobile application designed for anime enthusiasts to discover seasonal releases and manage their personal watchlists. Developed for the Pixel 3a running Android 15 (API 35), the app leverages the Jikan API to provide up-to-date information on currently airing shows, including titles, synopses, and cover art.

The motivation behind this project is to provide a clean, modern interface for tracking anime without the clutter of larger social platforms. It serves as a practical tool for users to quickly check what’s airing and save favorites for later viewing.

Technically, the application adheres to Google’s Modern Android Development (MAD) guidelines. It implements the MVVM architecture to ensure separation of concerns and state preservation across configuration changes, such as screen rotation. By using Retrofit for API communication, Room for local storage of favorites, and DataStore for persistent settings, the app avoids deprecated practices like Volley and SharedPreferences. This project demonstrates a robust understanding of the Android lifecycle and efficient resource management, ensuring a smooth and reliable user experience for students and casual fans alike.
